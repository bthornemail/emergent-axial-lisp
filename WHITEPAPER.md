## Bootstrap Claim

Emergent Axial Lisp preserves canonical program/data structure as S-expressions built from:

```text
NIL
ATOM
PAIR
```

This repository currently implements only that pair kernel. Multi-axial resolution, scoped environments, recovery, reflection, and lowering are deferred.
